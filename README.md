## Rodando com Docker

1. (Opcional) copie o arquivo `.env.example` para `.env` e ajuste as variaveis.
2. Suba os containers:

```bash
docker compose up --build
```

1. Acesse:

- Frontend: <http://localhost:8080>
- Backend (Swagger): <http://localhost:8000/docs>
